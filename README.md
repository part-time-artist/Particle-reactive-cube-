# 🧊 Cube Particle System

A high-performance particle-based 3D cube visualization built with HTML, CSS, and React. This project demonstrates complex particle physics and rendering in a web environment.

## 🚀 Features
- **3D Particle Cube**: Fluid particle animations forming a geometric cube.
- **Responsive Design**: Adapts to various screen sizes.
- **Multiple Implementations**: Includes both standard React and Framer-based approaches.

## 🛠️ Tech Stack
-   **Structure**: HTML5
-   **Styling**: Vanilla CSS
-   **Logic**: JavaScript / TypeScript
-   **Frameworks**: React, Framer Motion (optional)

## 📁 Project Structure
- `preview_cube fin fin.html`: Main entry point for the vanilla implementation.
- `FramerPointCloudCube.tsx`: Advanced React-based particle system.
- `StandardReactCube.tsx`: Optimized standard React implementation.
- `script.js` & `style.css`: Core logic and styling for the vanilla version.

## 🏁 Getting Started
Simply open `preview_cube fin fin.html` in your browser to see the visualization in action!
